HRD 연동 방법

  1. 하베스터 다운로드 후 압축을 풀어 한 폴더에 둡니다(아래 버튼).
  2. 최초 1회: pip install mitmproxy 설치.
  3. start-harvester.bat 실행 → 브라우저로 http://mitm.it 접속해 Windows 인증서(CA) 설치.
  4. Windows 프록시를 127.0.0.1:8899 로 지정.
  5. HRD 행정프로그램에 로그인하고 출결을 한 번 조회 → 자동 연동.
⬇ 하베스터 다운로드 (zip)
